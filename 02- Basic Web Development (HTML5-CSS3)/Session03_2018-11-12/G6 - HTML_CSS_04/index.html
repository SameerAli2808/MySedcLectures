Introduction

Introduction

  • Images are essential part of the web today
  • Together with text and hyperlinks, images help in enriching the content of the website
  • There are several things to consider when selecting and preparing images for your site, but taking time to get them right will make it look more attractive and professional

Images can be used to set the tone for a site in less time than it takes to read a description.

Storing images on your site

  • If you are building a site from scratch, it is good practice to create a folder for all of the images the site uses

Adding images

Adding images

  • <img />
  • Required Attributes: src and alt
  • More info: http://www.w3schools.com/tags/tag_img.asp

Example


<img src="img/imagename.png" alt="image description" />

Height & Width

  • height - specifies the height of the image in pixels
  • width - specifies the width of the image in pixels

Create image links

  • You can easily create link around an image by wrapping it inside Hyperlink anchor tag


<a href="link-here"/><img src="img/image.png" alt="image description"/></a>

Create Image Links -Thumbs

  • Same as you create Image links, you can create image with thumbnails
  • The only thing you should know here is that you should either scale the image to have two different sizes or have two different images, one for the thumbnail image and other for the original sized image

Choosing the right format

Image formats

  • JPEG
  • Whenever you have many different colors in a picture you should use a JPEG
  • Photographs

Image formats

  • GIF and PNG
  • When a picture has an area that is filled with exactly the same color, it is known as flat color
  • Logos, illustrations, and diagrams often use flat colors

Image formats

  • Read more on image formats: http://www.1stwebdesigner.com/different-image-formats/

Image optimization

Image dimensions

  • The images you use on your website should be saved at the same width and height that you want them to appear on the page
  • When cropping images it is important not to lose valuable information
  • It is best to source images that are the correct shape if possible

Image resolution

  • JPGs, GIFs, and PNGs belong to a type of image format known as bitmap
  • Images created for the web should be saved at a resolution of 72 ppi
  • The higher the resolution of the image, the larger the size of the file

Vector images

  • Vector images differ from bitmap images and are resolution-independent
  • Vector images are created by placing points on a grid, and drawing lines between those points
  • The advantage of creating line drawings in vector format is that you can increase the dimensions of the image without affecting the quality of it.

Scalable Vector Graphics (SVG)

  • SVG is used to define graphics for the Web
  • SVG is a W3C recommendation - <svg> introduced in HTML5
  • SVG has several methods for drawing paths, boxes, circles, text, and graphic images
  • w3c page for svg: http://www.w3schools.com/html/html5_svg.asp

Image Map

  • Image maps are a bit more complex than simple images
  • To create image map, you can use map tag with specifying one or more area tags combined with img tag which is linked to the map
  • w3c page for image maps: http://www.w3schools.com/tags/tag_map.asp

Image Map


<img src="img/image.png" width="145" height="126" 
alt="image" usemap="#imagemap" />

<map name="imagemap">
	<area shape="rect" coords="0,0,82,126" href="link1.html" alt="Shape1" />
	<area shape="circle" coords="90,58,3" href="link2.html" alt="Shape2" />
	<area shape="circle" coords="124,58,8" href="link3.html" alt="Shape3" />
</map>
