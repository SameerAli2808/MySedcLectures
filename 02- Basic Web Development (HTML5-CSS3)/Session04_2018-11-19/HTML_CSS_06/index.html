Introduction

Introduction

  • Traditionally, the term 'form' has referred to a printed document that contains spaces for you to fill in information
  • HTML borrows the concept of a form to refer to different elements that allow you to collect information from visitors to your site

The best known form on the web is probably the search box that sits right in the middle of Google's homepage

Build a form

Building a form

  • You add a form to an HTML page using the <form></form> tags

Form Controls

There are several types of form controls that you can use to collect information from visitors to your site

  • ADDING TEXT:
    • Text input
    • Password input
    • Text area (multi-line)

Form Controls

  • MAKING CHOICES:
    • Radio button
    • Checkboxes
    • Drop-down boxes

Form Controls

  • SUBMITTING FORMS:
    • Submit buttons
    • Image buttons

Form Controls

Form Structure

Form Structure

  • <form>
  • The <form> element should always carry the action attribute and will usually have a method and id attribute too

More on forms here: http://www.w3schools.com/html/html_forms.asp

Basic Form Structure


<form action="subscribe.php" method="get">
	<p>This is where the form controls will appear.
	</p>
</form>

Text input

Text input


<form action="login.php">
	<p>
		<label for="username">Username:</label>
		<input type="text" id="username" name="username" />
	</p>
</form>

Password input

Password input


<form action="login.php">
	<p>
		<label for="username">Username:</label>
		<input type="text" id="username" name="username" />
	</p>
	<p>
		<label for="username">Password:</label>
		<input type="password" id="password" name="password" />
	</p>
</form>

Text area

Text area


<form action="comments.php">
	<p>
	<label for="comments">What did you think of this gig?</label>
	</p>
	<textarea id="comments" name="comments" cols="20" rows="4">
		Enter your comments...
	</textarea>
</form>

Radio Button

Radio Button


<form action="profile.php">
	<p>Please select your favorite genre:</p>
	<p>
		<input type="radio" name="genre" value="rock"
		checked="checked" /> Rock
		<input type="radio" name="genre" value="pop" /> Pop
		<input type="radio" name="genre" value="jazz" /> Jazz
	</p>
</form>

Checkbox

Checkbox


<form action="profile.php">
	<p>Please select your favorite music service(s):</p>
	<p>
		<input type="checkbox" name="service"
		value="itunes" checked="checked" /> iTunes
		<input type="checkbox" name="service"
		value="lastfm" /> Last.fm
		<input type="checkbox" name="service"
		value="spotify" /> Spotify
	</p>
</form>
