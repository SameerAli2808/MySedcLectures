Seconds Calculator

Calculate the hours, minutes and seconds

Write a function named calculateTime that:

  • takes 1 arguments: seconds
  • calculates the hours, minutes and seconds from the input.
  • outputs the result to the screen like so: "There are 3 hours, 24 minutes and 32 seconds"
  • Call the function three times with different sets of values.
